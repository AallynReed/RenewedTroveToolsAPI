This object represents a leaderboard entry
#### **Structure**

- Fields
    - `player_name` > `String` - Name of the player
    - `rank` > `Integer` - Rank placement the player is in the leadeboard
    - `score` > `Float` - Score of the player for the leaderboard

#### **Example**
```json
{
    "player_name": "karelm",
    "rank": 1,
    "score": 187060.0
}
```
